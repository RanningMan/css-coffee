The 'grid-auto-rows' CSS property specifies the size of rows in a grid container when the rows are created automatically. This property allows you to define a consistent height for any rows that are added to the grid, ensuring a uniform layout. It can take various units, such as pixels or fractions, enabling flexible design options.
