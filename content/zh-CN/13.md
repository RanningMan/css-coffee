    animation-name: slidein; 
    animation-duration: 3s; 
} 
@keyframes slidein { 
    from { margin-left: 100%; } 
    to { margin-left: 0%; } 
}"
---

'animation-name' 属性指定要应用于选定元素的关键帧动画的名称。通过引用关键帧动画，您可以定义动画的行为和样式属性。
