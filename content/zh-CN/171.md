  .container {
    width: 200px;
    border: 1px solid #ccc;
    padding: 10px;
    overflow-wrap: break-word;
  }

  .hyphenated {
    hyphens: auto;
    font-size: 16px;
    line-height: 1.5;
    width: 100%;
    overflow-wrap: break-word;
  }
---

CSS 中的 'hyphens' 属性控制文本块中单词的自动连字符，当它们超出容器时。这个属性可以通过在适当的位置拆分长单词并在必要时添加连字符，从而提高狭窄文本列的可读性。它对于有连字符规则的语言特别有用。
