  background-color: red;
  animation: slide 2s infinite; /* Example animation */

@keyframes slide {
  from {
    transform: translateX(0);
  }
  to {
    transform: translateX(200px);
  }
}"
---

CSS中的'animation-range'属性允许您指定动画迭代的值范围。这个范围是通过'from'和'to'关键字定义的，从而实现指定值之间的平滑过渡。<br />
